# PracticeHub – Competitive Exam Practice & Consistency Tracker

PracticeHub is a MERN-based platform for competitive exam aspirants (JEE/NEET/SSC) inspired by LeetCode style practice systems.

## Features
- Login / Signup with Exam Selection
- JEE Practice Flow: Subject → Chapter → Quiz
- Timed MCQ practice (90 sec/question)
- Daily Tracker Calendar (LeetCode/GitHub style)
- Streak system & progress analytics
- Founder page: Sachin Mishra

## Tech Stack
**Frontend:** React.js, Tailwind CSS, React Router, Axios  
**Backend:** Node.js, Express.js, JWT, bcrypt  
**Database:** MongoDB (Mongoose)
